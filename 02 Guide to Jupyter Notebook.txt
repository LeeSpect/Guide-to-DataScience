Ctrl + Enter = 셀 실행
b = 아래에 셀 추가
a = 위에 셀 추가
dd = 셀 삭제

Alt + Enter = 셀 실행 후 아래에 셀 생성
Shift + Enter = 셀 실행 후 아래 셀 선택 / 셀 실행 후 (아래에 셀이 없다면) 아래에 셀 추가

Kernel + Restart & Clear Output = 새로운 세션을 시작
Kernel + Restart & Run All = 새로운 세션을 시작하고, 모든 셀을 순서대로 실행


Cell type을 Code에서 Markdown으로 변경하면 설명글 작성 가능

- 제목
#를 사용하여 단계별로 제목을 표현할 수 있습니다.
#을 하나만 쓰면 1단계 제목이고, # 을 4개 사용하면 4단계 제목이 됩니다. 단계가 높아질수록 글자 크기가 작아집니다.

# 가장 큰 제목
## 두 번째 큰 제목
### 세 번째 큰 제목
--------------------------------------------------------------------------------------------------------------
- 번호가 있는 목록
아래처럼 적으면 번호가 있는 목록(리스트)을 만들 수 있습니다.
1. 첫 번째 할일입니다. 
2. 두 번째 할일입니다.
3. 세 번째 할일입니다.
--------------------------------------------------------------------------------------------------------------
- 번호가 없는 목록
숫자 대신 *을 사용하면, 번호가 없는 목록을 만들 수도 있습니다.
* list item 1
* list item 2
* list item 3

-> ・list item 1
-> ・list item 2
-> ・list item 3
