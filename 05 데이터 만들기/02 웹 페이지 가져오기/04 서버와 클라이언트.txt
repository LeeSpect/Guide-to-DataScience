# 05 데이터 만들기
# 0502 웹 페이지 가져오기
# 050204  서버와 클라이언트

웹에는 서버(Server)와 클라이언트(Client)라는 것이 있습니다.
서버는 서비스를 제공하는 쪽이고, 클라이언트는 서비스를 제공받는 쪽입니다.

페이스북 웹사이트에 들어가려면?
우선 크롬이나 인터넷 익스플로어 같은 브라우저를 실행하는데 바로 이 브라우저가 웹 클라이언트입니다.
즉 웹 페이지를 제공받으려는 고객인 셈입니다.

주소창에 페이스북의 주소를 치면 웹 클라이언트는 웹 서버에게 페이스북 웹 페이지에 해당되는 내용을 요청(Request)합니다.
웹 서버는 요청을 받고 필요한 것들을 응답(Response)합니다.

웹 서버는 웹 클라이언트에게 정확히 어떤 것을 Response할까요?
우선 가장 기본적으로는 웹 페이지에 해당되는 HTML을 제공합니다.
그리고 필요한 CSS 코드나 JavaScript 코드에 대한 요청이 들어오면 그것에 대한 것들도 돌려 줄 수 있습니다.

웹 클라이언트인 브라우저는 웹 서버로부터 받은 코드들을 통해 웹 페이지를 사용자에게 보여줄 수 있게 되는 겁니다.
